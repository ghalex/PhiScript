PhiScript for Javascript

PhiScript is a JavaScript framework that provides essential functionalities for
building cross-browser web applications.

Overview About, Dependencies, Roadmap

About

PhiScript main goal is to provide every essential functionality you need in the process of building web applications using Javascript. With PhiScript you can build your application layout and user interface very fast and easy. For now PhiScript is in development stage but you can download an alpha version and play with it right away from here:

Download (PhiScript Alpha 0.5.2) (non-commercial use only)

To learn more on how to use PhiScript and what UI Components are available for now please see sections bellow.

Browser support

PhiScript is tested and supported in major modern browsers like Chrome, Safari, Internet Explorer, and Firefox.

  • Firefox 5+
  • Latest Google Chrome
  • Latest Safari
  • Internet Explorer 8+

Dependencies

PhiScript is build using Mootools so to used you need to download Mootools first. If you don't know what Mootools is or never used before please visit Mootools website.

Layout RootBox, CellBox, HBox, VBox, HTMLBox

Using layout containers

With PhiScript you can build your application layout very fast and easy using layout containers. For most of the applications the HBox and VBox containers is all you need to build your layout, but PhiScript also offers you other layout containers. Here is the list of containers that PhiScript offers for now:

  • RootBox - this is the main container, it's element is <body>
  • CellBox
  • HBox
  • VBox
  • HTMLBox

Let's try to build a layout and I will explain more on how to use each container. In this example we will try to build this layout:

The code to build that layout looks like this: 

var mainVBox = new Phi.UI.VBox(); // this is main vbox
var bottomRightVbox = new Phi.UI.VBox(); 

mainVBox.setWidth("100%");
mainVBox.addChild( new Phi.UI.Text('1'));
mainVBox.addChild( new Phi.UI.HBox().addChild( new Phi.UI.Text('2'))
				    .addChild( bottomRightVbox ));

// and now we add another HBox for 3 and 4
bottomRightVbox.addChild( new Phi.UI.HBox() );
bottomRightVbox.addChild( new Phi.UI.Text('5'));

// get top HBox and add more childs to it
bottomRightVbox.firstChild().setWidth("100%")
			    .setHorizontalAlign( HorizontalAlign.CENTER )
			    .addChild( new Phi.UI.Text('3'))
			    .addChild( new Phi.UI.Text('4')));

As you can see you can combine layout containers very easy to obtain the result you want. The addChild returns this so you can used in chain. For the demo I will use buttons instead of Phi.UI.Text and to inject the result in a selected div I will use: 

var htmlBox = new HTMLBox('layoutDemoDiv');
htmlBox.addChild( mainVBox );

Demo

Here is a list of functions you can use for a HBox or VBox:

  • setHorizontalAlign( align ) - sets all childs horizontal align
  • setVerticalAlign( align ) - sets all childs vertical align
  • setGap( gapValue ) - sets the distance between children
  • setCellWidth(index, value), setCellHeight(index, value) sets width/height for a specific cell
  • setChildVerticalAlign(child, align) - sets vertical align only in a specific cell, child can be an index or a child
  • setChildHorizontalAlign(child, align) - sets horizontal align only in a specific cell

Another important interface you must know is ChildList with the following functions:

  • addChild( child)
  • addChildAt( child, index )
  • getChildAt(index)
  • getChildIndex( child )
  • lastChild()
  • firstChild()
  • removeChild( child )
  • hasChild( child )
  • removeChildAt( index )
  • getChildren()
  • removeAllChildren()

This functions are available to all classes that extend from Container like RootBox, CellBox, HBox, VBox, HTMLBox

Road map

Planned containers for version 1.0.0:

  • CanvasBox - display children at a specific position
  • HDivideBox - similar to HBox but inserts a divider between each child
  • VDivideBox - similar to VBox but inserts a divider between each child

User Interface Button, CheckBox, TextInput, List

User Interface components

Now that you know how to build your application layout very fast and easy is time to see how you can capture user input and used. To get user input PhiScript offers for now basic UI componenets like Button, CheckBox, TextInput and List. Let's see a demo and some code :

Demo

And the code looks like this: 

var uiHTMLBox = new Phi.UI.HTMLBox('uiDemoDiv');
var uiHBox = new Phi.UI.HBox();
var txtInput = new Phi.UI.TextInput();
var sayBtn = new Phi.UI.Button("Say");
var clearBtn = new Phi.UI.Button("Clear")
var chk = new Phi.UI.CheckBox();

// Add custom class to UI components
sayBtn.addClass('primary');
			
// Layout
uiHBox.setGap(2)
      .addChild(new Phi.UI.Label("Your name: "))
      .addChild(txtInput)
      .addChild(sayBtn)
      .addChild(clearBtn)
      .addChild(chk)
      .addChild(new Phi.UI.Label("Name valid"));
				  
			
uiHTMLBox.addChild( uiHBox );

// Now we do some user interaction
sayBtn.addEvent('click', function() { alert('Hello: ' + txtInput.getText()) });
clearBtn.addEvent('click', function() { txtInput.setText("")});

txtInput.addEvent('change', 
	function() 
	{ 
		if(txtInput.getText().length > 0)
			chk.setChecked( true );
		else 
			chk.setChecked( false );
	}
);
